Proyecto React (TypeScript) generado a partir del ZIP original.

Estructura:
- src/components: Navbar, Footer, ProductCard
- src/pages: Index, Product, Contact
- src/data/products.ts: productos detectados (o por defecto)
- public/index.html
- package.json, tsconfig.json

Instrucciones rápidas:
1. Descomprime 'pasteleria_react_converted.zip'
2. En la carpeta del proyecto ejecuta:
   npm install
   npm run dev

Nota: Revisar src/assets para imágenes copiadas del ZIP. Ajusta estilos en src/index.css
